# 第 1 章 Cursor 簡介與安裝

## 1-2 安裝 Cursor

### 下載並安裝 Cursor

- 透過 winget 安裝的指令：

    ```
    winget show Anysphere.cursor
    ```

- 透過 scoop 安裝的指令：

    ```
    scoop bucket add extras
    scoop install cursor
    ```

- 透過 homebrew 安裝的指令：

    ```
    brew install --cask cursor
    ```

- Linux 下的安裝指令：

    ```
    sudo dpkg -i cursor-2.1-linux-x64.deb
    ```

## 1-3 牛刀小試--安裝管理 Python 環境的 uv 工具

### 指揮 Agent 幫我們安裝工具

#### 在 Windows 上使用 scoop 建置工具鏈

- 輸入的提示內容：

    ```md
    幫我安裝套件管理工具，Windows 請安裝 scoop，其他平台安裝 homebrew
    ```

    > [!TIP]
    > 萬一 Agent 無法幫你正確安裝 scoop，你也可以自行在終端機執行以下指令安裝：
    > ```
    > Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
    > Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression
    > ```

- 安裝 uv 工具：

    ```
    我已經安裝好 scoop，用 scoop 安裝 uv
    ```

- 安裝 Node.js：

    ```
    再用 scoop 安裝 nodejs
    ```

- 驗證安裝結果：

    ```
    驗證 uv 和 nodejs 都安裝成功
    ```

- 安裝 PowerShell Core：

    ```
    用 scoop 安裝 powershell core
    ```

#### 在 macOS/Linux 上使用 Homebrew 建立工具鏈

- 輸入的提示內容：

    ```md
    幫我安裝套件管理工具，Windows 請安裝 scoop，其他平台安裝 homebrew
    ```
- 在終端機按 <kbd>Ctrl</kbd>+<kbd>K</kbd> 輸入的提示內容：

    ```
    安裝 homebrew
    執行剛剛安裝步驟最後面要我加入路徑的指令
    驗證 homebrew 安裝完成
    ```

- 安裝 uv 工具：

    ```
    我已經安裝 homebrew，用 homebrew 幫我安裝 uv
    ```

- 安裝 Node.js 工具：

    ```
    再幫我 安裝 nodejs
    ```

- 驗證安裝結果：

    ```
    驗證剛剛 uv 和 nodejs 都有安裝好
    ```
